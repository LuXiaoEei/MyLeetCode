# Longest Common Prefix

寻找最长的公共前缀

一个一个比对,通过查看集合的长度是否为1

```py
class Solution:
    def longestCommonPrefix(self, strs):
        """
        :type strs: List[str]
        :rtype: str
        """
        if len(strs)==0:
            return ''
        
        Min=min([len(x) for x in strs])
        if Min==0:
            return ''
        
        if len(set([x[0] for x in strs]))>1:
            return ''
        
        for i in range(Min):
            if len(set([x[i] for x in strs]))>1:
                return strs[0][:i]
        return strs[0][:Min]
                
            
```
